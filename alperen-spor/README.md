# Alperen Spor

## Overview
Alperen Spor is a web application designed to provide a comprehensive platform for managing sports-related activities. This project is built using Next.js and TypeScript, ensuring a modern and efficient development experience.

## Features
- **Global Error Handling**: The application includes a global error component to provide user-friendly error messages.
- **Dynamic Layout**: A flexible layout component that defines the overall structure of the application.
- **Authentication**: Components dedicated to user authentication, ensuring secure access to the platform.
- **Dashboard**: A dedicated section for users to manage their activities and view statistics.
- **API Integration**: Functions for making API calls to a backend service, including integration with Supabase for database management.
- **Responsive Design**: The application is styled with global CSS to ensure a consistent look and feel across all devices.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd alperen-spor
   ```
3. Install dependencies:
   ```
   npm install
   ```
   or
   ```
   yarn install
   ```

### Running the Application
To start the development server, run:
```
npm run dev
```
or
```
yarn dev
```
The application will be available at `http://localhost:3000`.

### Deployment
Refer to the [DEPLOYMENT.md](DEPLOYMENT.md) file for instructions on how to deploy the application to a production environment.
